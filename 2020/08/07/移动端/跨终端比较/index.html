跨终端比较

刘小军 2020-08-07 15:31:53

原生开发

Java、C++、Objective-C。
支持在线或离线，消息推送或本地资源访问，摄像拨号功能的调取。

优点

  • 最佳用户体验，流畅
  • 速度更快
  • 几乎所有功能都能实现.

痛点

  • 实时性限制&&新功能依赖升级
  • Bug无法热修复
  • 多端协作成本高，不能跨平台

跨平台开发

跨平台开发主流技术

  • Flutter（Google）
    • Dart，
  • ReactNative（FaceBook）
    • React
  • Weex（Alibaba）
    • Vue （生态一般）
  • Hybrid App
    • 原生 + 网页
  • Cordova（原PhoneGap，Adobe）（）
  • 小程序，快应用
